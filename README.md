## ğŸ†“ Free Deployment Guide

If you want to host this app online for free so you can share it, choose one of these three methods:

### Option A: Vercel (Easiest)
1. Go to [Vercel.com](https://vercel.com) and sign up for a free account.
2. Click **"Add New"** > **"Project"**.
3. If your code is on GitHub, import the `calcraft` repo. 
4. Vercel will automatically detect **Vite** settings. Click **Deploy**.
5. Your site will be live at `calcraft.vercel.app`.

### Option B: Netlify (No Git Required)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Simply **Drag and Drop** your entire project folder into the browser.
3. Netlify will host it instantly. You can change the random URL in "Site Settings".

### Option C: GitHub Pages (Automatic)
*Note: This is already configured in this project via `.github/workflows/deploy.yml`.*
1. Create a repository named `calcraft` on GitHub.
2. Upload all your files (you can use the "Upload files" button on GitHub.com).
3. Go to **Settings > Pages**.
4. Set **Source** to **"GitHub Actions"**.
5. Your site will automatically build and go live.
